# Suggested commands
- Start local static server from repo root: `python -m http.server 4173`
- Quick JS syntax check: `node --check script.js`
- List files on Windows PowerShell: `Get-ChildItem -Force`
- Search text fast: `rg -n "pattern" index.html products.html script.js style.css`
- Open the local site after starting server: `http://127.0.0.1:4173/index.html`
- Manual responsive/product-page verification: test `index.html`, `products.html`, and `products.html?product=vmc-850` in browser.
