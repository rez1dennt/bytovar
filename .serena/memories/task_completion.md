# When a task is completed
- Run `node --check script.js` after JavaScript edits.
- Serve the site locally with `python -m http.server 4173` and manually verify both pages in a browser.
- Check key flows: header/menu behavior, anchor navigation on landing page, transition to `products.html`, filter chips, product detail opening through URL params, and mobile layout.
- Review browser console for runtime errors after a fresh page load.
